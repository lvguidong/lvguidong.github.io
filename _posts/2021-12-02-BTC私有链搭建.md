# 配置文件

在文件写下`bitcoin.conf`

```tex
regtest=1

rpcallowip=127.0.0.1
rpcuser=lvguidong
rpcpassword=lvguidong

server=1
daemon=1
txindex=1
```

写一个启动脚本`run-bitcoin.sh`

```
#!/bin/bash

basedir=`pwd`
bitcoind  -conf=$basedir/conf/bitcoin.conf -rpcbind=127.0.0.1 -datadir=$basedir/data
```



# 启动regtest私链

```shell
sh run-bitcoin.sh
```

在上述的data目录下有数据了，产生的区块信息和钱包相关的信息都保存在这个目录下。
